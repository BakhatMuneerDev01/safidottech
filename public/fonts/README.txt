Place your font files here: public/fonts/

Required files (download from sources below):

CLASH DISPLAY — https://www.fontshare.com/fonts/clash-display
  ClashDisplay-Medium.otf      (weight 500)
  ClashDisplay-Semibold.otf    (weight 600)
  ClashDisplay-Bold.otf        (weight 700)

SATOSHI — https://www.fontshare.com/fonts/satoshi
  Satoshi-Regular.otf          (weight 400)
  Satoshi-Medium.otf           (weight 500)
  Satoshi-Bold.otf             (weight 700)

JETBRAINS MONO — https://www.jetbrains.com/lp/mono/
  JetBrainsMono-Regular.otf    (weight 400)

NAMING CONVENTION: Exact FamilyName-Weight.otf as shipped by Fontshare.
Do NOT rename files — the @font-face src paths match these names exactly.

NOTE: Files in public/ are served at root with stable (non-hashed) URLs,
which is required for <link rel="preload"> to work correctly.

FUTURE UPGRADE: If you convert to WOFF2 for smaller file sizes,
update fonts.css format("opentype") → format("woff2")
and rename files to .woff2. No other changes needed.
